# cs61b_2021sp
课程网页:https://sp21.datastructur.es/
学习时间:2025.11.23->2026.02.06

## 课程内容:
1. java基础
2. 数据结构:
- 列表:链表,动态数组
- 树:二叉树,2-3-4树,红黑树
- 哈希表
- 堆与优先队列
- 图
3. 算法
- 排序算法:选择排序,插入排序,归并排序,快速排序,堆排序
- 图算法:深度优先搜索,广度优先搜索,最短路径算法,最小生成树算法
- 渐进分析
4. 软件工程实践
